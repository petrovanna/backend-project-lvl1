### Installation:

Make sure you have Node.js version 13 or higher

git clone git@github.com:petrovanna/backend-project-lvl1.git

npm ci

npm link

### Run game:

Enter the command of the game you want to play:

brain-even

brain-prime

brain-progression

brain-gcd

brain-calc
